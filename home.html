PB-QUANTS

An R-Shiny based webpage for analysis of plant breeding experiments. PB-QUANTS simplifies statistical analysis in R bridging frontend graphical user interface with R optimized scripts running in the backend. Hence scientist can achieve the analytical robustness and accuracy of various packages in R without having to type a single line of code. Also, PB-QUANTS provides simple graphical workflow for different data analytics starting from importing the data to proving the end results in MS-EXCEL format with publication standard tables and images.




PB-QUANTS uses variuos dependencies such as agricolae, plantbreeding, corrplot, ggplot and various other R packages for carrying out analysis.
